# Contributing

Thanks for your interest! This is a hobby project, but contributions are welcome.

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes and test: `make test`
4. Format code: `make fmt`
5. Submit a PR

## Code Style

- Run `gofmt` before committing
- Use typed structs for API payloads
- Never commit secrets or personal data

## Reporting Issues

Open an issue with:
- What you tried to do
- What happened
- Your environment (OS, Go version)

## Security

Found a vulnerability? Please report privately - see [SECURITY.md](SECURITY.md).
